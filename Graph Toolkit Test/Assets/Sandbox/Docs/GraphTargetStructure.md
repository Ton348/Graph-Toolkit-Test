# Graph Target Structure

## GraphCore
- Editor — общий editor-слой Graph Toolkit
- Runtime — общий runtime-слой Graph Toolkit
- Server — общий серверный/контрактный слой Graph Toolkit

## Game1Graph
- Editor — editor-ноды и authoring первой игры
- Runtime — runtime-ноды и интеграции первой игры

## Game2Graph
- Editor — editor-ноды и authoring второй игры
- Runtime — runtime-ноды и интеграции второй игры

## Sandbox
- Docs — временная документация по миграции
- Bootstrap — запуск и инициализация sandbox
- Debug — debug-инструменты
- Mocks — mock-реализации и временные заглушки
- Scenes — тестовые сцены
